# 🚀 Battlefield 6WD Rover

A rugged and intelligent 6-wheel-drive (6WD) battle rover powered by the ESP32 microcontroller, featuring advanced motor control, obstacle avoidance, and off-road capability. Built for robotics competitions and real-world terrain exploration.

---

## 🔧 Features

- 🧠 Controlled by ESP32 microcontroller
- ⚙️ 6WD configuration using BTS7960 motor drivers
- 🧲 Obstacle avoidance with ultrasonic sensor
- 🛰️ IMU-based orientation sensing (MPU6050)
- 📡 Remote/manual control ready (Bluetooth or Wi-Fi)
- 🔋 Li-ion battery powered with efficient distribution
- 📷 Expandable for camera or sensor add-ons

---

## 🛠️ Hardware Used

| Component            | Quantity | Description                         |
|---------------------|----------|-------------------------------------|
| ESP32 Dev Module    | 1        | Microcontroller (Wi-Fi + BT)        |
| BTS7960 Motor Driver| 3        | Dual-channel H-Bridge, 43A          |
| BO Gear Motors      | 6        | Drives wheels                       |
| Ultrasonic Sensor   | 1        | HC-SR04 for obstacle detection      |
| MPU6050             | 1        | IMU sensor for motion/orientation   |
| Li-ion Battery Pack | 1        | 2S or 3S, 7.4V–11.1V                |
| Custom 6WD Chassis  | 1        | Metal frame with mounting space     |
